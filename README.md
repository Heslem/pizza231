## SWOT-анализ для проекта

Работа с git
```
запустите Git Bash 
перейдите в каталог c:/xampp/htdocs
> cd c:/xampp/htdocs
выполните в bash терминале
> git pull
```
Выполните работу по курсу "Внедрение":  
1. SWOT-анализ для курсового проекта в виде таблицы с именем swot.docx   
2. Описание целевой аудитории для своего курсового проекта в том же файле.  
3. Сохраните файлв папку проекта /storage  

Закоммитьте и запуште изменения
```
> git status
> git add .
> git status
> git commit -m "SWOT анализ и целевая аудитория"
> git push
```
